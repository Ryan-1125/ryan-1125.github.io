一场奔赴，一半是热爱，一半是情怀。我们之所以奔赴山城重庆，源于两张李荣浩“黑马”巡回演唱会的门票——恰逢他的巡演落地重庆，我们皆是他的歌迷，又因朋友的老家在重庆，我高二时也曾与家人同游过这座山城，所以对山城，我们都藏着一份特别的牵挂与眷恋。

2025年10月31日，周五下课以后，我们从各自的学校出发，奔赴广州白云机场，搭乘飞机前往山城重庆。
![alt text](9293557a0429e8301c0855c33a7050e6.jpg)

等到落地时，夜色已漫过山城，华灯初上，山城的温柔与热闹，在夜色里慢慢铺展开来。

# 夜巡：旧忆与新景的碰撞

我们的酒店在渝中区解放碑附近，放下行李，便迫不及待走出酒店，彼时天飘着零星小雨，细碎又温柔，为山城的夜晚添了几分朦胧感。

先去了解放碑，看着眼前熟悉又陌生的景象，心中满是感慨。上次来还是高二盛夏的白天，现今是大学周末的雨夜。时隔数年，再次站在这里，游人依旧，灯火依旧，只是身边多了合拍的好友，多了几分不一样的心境。
![alt text](2e68c7893e17aca5f35c83559d9fa0a7.jpg)

![alt text](bbb7dd8b6cade4f2b15b6c92f6674ac1.jpg)
随后前往洪崖洞，洪崖洞的最佳观景位在江对岸，而当时已经比较晚了，我们径直走到了洪崖洞的顶上。抵达时，恰好赶上灯光熄灭，恰逢淡季，游人稀少，没有了亮灯时的璀璨喧嚣，熄灯后的洪崖洞多了几分静谧古朴，别有一番韵味——人们见到的都是它灯火辉煌的模样，这般灯火阑珊的它，或许也有动人之处。

从洪崖洞下来左转，我牵着朋友边走边念叨：“我高二和家里人来的时候，曾在重庆的一栋楼里坐电梯上到二十楼，出来还是平地，那种感觉我一直记着，特别神奇。” 说着，我努力回想当时的路线、周边的模样，细碎的记忆片段一点点拼凑，比对着眼前的建筑，心底那份隐约的感觉一直指引着我们往前走。我们逛了约莫一会儿，眼前突兀现此屋——它的轮廓和我记忆里的模样渐渐重合，就是这栋，应该就是它！我们走进大厦，按下了高处的电梯按钮。

电梯门打开的那一刻，和当年我看到的景象一模一样。雨夜的平台上，周边建筑的灯光零星闪烁，湿润的空气里裹着山城的气息，我和朋友站在原地，相视一笑，那种“居然真的找到了”的惊喜，还有当下的场景与三年前的记忆完美重合的恍惚感，让时空仿佛真的交叠在了一起。有些回忆从未褪色，那些不经意间记下的瞬间，总会以这样巧合的方式重新出现在眼前。

之后，我们一路步行返回酒店。雨已经停了，地面还带着湿润的水汽，山城的灯光洒在路面上，映出细碎的光斑。路边偶尔能看到几栋西洋风格的小洋楼，在夜色中格外有格调。人不多，晚风轻柔，一步步走着，没有匆忙，只有满心的浪漫与惬意。
![alt text](a817fdc459fa387cd3a3fb74332469df.jpg)

# 漫游：我又回到那老街

第二天，我们过江前往南岸，第一站便去了长江国际——那是一栋高耸挺拔的双塔建筑，立面简约大气，透着现代都市的质感。我们乘电梯上到高处，站在观景处眺望，江景尽收眼底，江水悠悠流淌，两岸的山城风貌错落有致，透过玻璃与晨雾，山城的江景壮阔而平静。
![alt text](30ed0a440bc4abbb8ca7de3ff2553819.jpg)

后来我们往东北方向走去，抵达了下浩里，它与龙门浩老街紧紧相连，是一处藏在山间的文艺街区。这里古色古香，依山而建，青石板台阶错落有致，被小雨浸润后，更显温润。
![alt text](e7e48dbcee2fc41bee56686217028047.jpg)

街区里藏着不少小店和小吃摊，现代气息与古朴韵味完美融合，既有老山城的烟火气，又有适合年轻人的小情调，逛起来轻松又惬意，格外适合我们这样的大学生。
![alt text](c4abcdf1edd1b322f4695342f753d9d5.jpg)

龙门浩老街脚下是热闹的集市，抬头便看到一块写着“老街”的牌匾，那一刻，李荣浩的《老街》旋律不自觉在耳边响起，“一张褪色的照片，好像带给我一点点怀念”，旋律与眼前的景象完美契合。
![alt text](016575600b5bf8fbdc9413bfdd520d86.jpg)

午后，我们转移了阵地——换到了奥体中心旁边的酒店，离体育馆近得离谱，步行几步就能到达，为晚上的演唱会省去了不少麻烦。

# 黑马：我从远方而来，赴你一面之约

夜幕降临，我们怀着满心期待，赴这场期待已久的李荣浩“黑马”演唱会。

演唱会开篇，舞台灯光骤然暗下，前奏响起，正是专辑同名主打歌《黑马》，李荣浩缓缓升上舞台，眼睛蒙着一条红色丝带，有点帅炸了。
![alt text](f4f58d070b20ca46bfa22af0bfac664a.jpg)

我爱听的不遗憾、年少有为、模特，很喜欢现场版的演绎。
![alt text](5f7b9602efb149e229b4a510aaaf7b32.jpg)

“你浅浅的微笑就像乌梅子酱”，甜甜的旋律刚漫开，右后方突然传来一阵欢呼——有人借着这首歌求婚了！距离我们很近，我们和周边的观众一起鼓掌祝福，温柔的旋律、热烈的欢喜，都定格成了最难忘的瞬间。
![alt text](dd4d0353b59cec2cec6397ab0f00a67b.jpg)

当《老街》的旋律响起，白天在龙门浩老街的所见所感瞬间涌上心头，“巷尾老爷爷卖的热汤面，味道弥漫过旧旧的后院”，熟悉的歌词，温柔的旋律，唱尽了对旧时光的怀念，也勾起了我高二时游山城的细碎回忆。
![alt text](4f10691a9ffd0d63cf6208a6e96d55a5.jpg)

为别人写的词，都是我们非常非常喜欢的歌，也是他的来时路。
![alt text](e02cc631e2e11b6b72497214b31a2c06.jpg)

戒烟、女儿国和紫荆花盛开。后两首算是风格很不一样的了。紫荆花盛开这首歌的时候全场都是粉紫色，真的绝美。
![alt text](f573acf115d02fb43238209453706b02.jpg)

小小的李荣浩↓   到燃曲的时候，鼓点一响全场沸腾，打鼓环节超惊艳，最后索性丢掉鼓棒了
![alt text](0443b838f7531cd0c26e50e6a331ef9e.jpg)

整场演唱会，时长两个多小时，没有多余的废话，全是满满的诚意，每一首歌曲，每一个舞台设计，每一次互动，都能感受到他的用心。他真的是一位全能音乐人，演唱、作词、编曲、奏乐，每一项都做到了极致。

演唱会结束的那一刻，灯光亮起，那种意犹未尽的感觉萦绕在心头。走出场馆，耳边依旧回荡着他的歌声。

# 尾声：

周日下午，我们踏上了返回深圳的飞机。飞行途中，幸运地碰见了一场绝美日落——太阳悬在远方，我们穿梭在云层之上。即将穿越云层的瞬间，整片云海都被染成了温柔的橙色，像被熔金包裹。
![alt text](86af95963f6ce5c3af0934ac9b5bd784.jpg)

这场山城之行，短暂却无比充实。它不是一场刻意的旅行，而是一场以歌为契的重逢。

于我而言，山城是藏着旧回忆的地方，高二的细碎时光，在重走的街巷里慢慢浮现；于我们而言，山城是盛满热爱的地方，朋友的故乡、李荣浩的旋律，都让这座山城更具温度。

我喜欢重庆魔幻的地形，喜欢它雨夜的浪漫，喜欢它的市井烟火，喜欢它藏在火爆里的温柔；更喜欢这场旅行里的每一份惊喜与感动——是旧忆重逢的感慨，是好友相伴的惬意，是奔赴演唱会的狂热，更是山城独有的治愈力量。

山城，我们后会有期。
