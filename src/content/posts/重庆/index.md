一场奔赴，一半是热爱，一半是情怀。我们之所以奔赴山城重庆，源于两张李荣浩“黑马”巡回演唱会的门票——恰逢他的巡演落地重庆，我们皆是他的歌迷，又因朋友的老家在重庆，我高二时也曾与家人同游过这座山城，所以对山城，我们都藏着一份特别的牵挂与眷恋。

2025年10月31日，周五，我们从各自的学校出发，奔赴广州白云机场，搭乘飞机前往山城重庆。一路满心期待，等到落地时，夜色已漫过山城，华灯初上，山城的温柔与热闹，在夜色里慢慢铺展开来。

（配图：广州白云机场随拍 / 飞机舷窗夜景 / 重庆机场抵达照）

# 山城夜巡：旧忆与新景的碰撞

我们的酒店定在渝中区解放碑附近，放下行李，便迫不及待走出酒店，彼时天飘着零星小雨，细碎又温柔，为山城的夜晚添了几分朦胧感。

先去了解放碑，看着眼前熟悉又陌生的景象，心中满是感慨。时隔数年，再次站在这里，人潮依旧，灯火依旧，只是身边多了合拍的好友，多了几分不一样的心境。

随后我们前往洪崖洞，世人都说，洪崖洞的最佳观景位在江对岸，而我们却径直走到了洪崖洞的顶上。抵达时，恰好赶上灯光熄灭，恰逢淡季，游人稀少，没有了亮灯时的璀璨喧嚣，熄灯后的洪崖洞多了几分静谧古朴，与亮灯时的震撼截然不同，别有一番韵味——我们都曾见过它灯火辉煌的模样，这般安静的它，反而更显动人。

从洪崖洞下来，我凭着高二时的记忆，牵着朋友往左手边的路走去。走了约莫一会儿，便找到了记忆中的那栋大厦（已然记不清名字，只记得熟悉的轮廓），我们一起走进电梯，按下高处的按钮。

电梯门打开的那一刻，熟悉的魔幻感扑面而来——别人的高空楼层是俯瞰城市的视角，而这里，走出电梯便是一片平坦的路面，仿佛穿越了空间的界限。雨夜的平台上，周边建筑的灯光零星闪烁，湿润的空气里裹着山城的气息，那一刻，时空仿佛交叠，高二时的记忆与当下的欢喜，紧紧相拥。

之后，我们一路步行返回酒店。雨已经停了，地面还带着湿润的水汽，山城的灯光洒在路面上，映出细碎的光斑。路边偶尔能看到几栋西洋风格的小洋楼，在夜色中格外有格调，人不多，晚风轻柔，一步步走着，没有匆忙，只有满心的浪漫与惬意。

（配图：解放碑夜景 / 熄灯后的洪崖洞 / 大厦电梯出口平台随拍）

# 山城漫游：烟火与诗意并存

第二天，我们过江前往南岸区，第一站便去了长江国际——那是一栋高耸挺拔的双塔建筑，立面简约大气，透着现代都市的质感。我们没有只停在楼下，而是走进大厦，乘电梯上到高处，站在观景处眺望，江景尽收眼底，江水悠悠流淌，两岸的山城风貌错落有致，风一吹，格外惬意，那一刻，才算真正体会到山城江景的壮阔与温柔。

（配图：长江国际外观 / 大厦内电梯随拍 / 江景远眺特写）

从长江国际出来，我们往东北方向走去，抵达了下浩里，它与龙门浩老街紧紧相连，是一处藏在山间的文艺街区。这里古色古香，依山而建，青石板台阶错落有致，被小雨浸润后，更显温润。

（配图：下浩里入口 / 青石板台阶特写 / 山间街区远景）

街区里藏着不少小店和小吃摊，现代气息与古朴韵味完美融合，既有老山城的烟火气，又有适合年轻人的小情调，逛起来轻松又惬意，格外适合我们这样的大学生。我们只是慢慢闲逛，看路边的老民居，尝一口地道的山城小吃，没有买任何东西，却也收获了满满的惬意与欢喜。

（配图：下浩里小店外景 / 路边小吃摊 / 我们闲逛的随拍）

走到龙门浩老街，脚下是热闹的集市，抬头便看到一块写着“老街”的牌匾，那一刻，李荣浩的《老街》旋律不自觉在耳边响起，“一张褪色的照片，好像带给我一点点怀念”，旋律与眼前的景象完美契合，温柔又治愈，仿佛所有的时光都慢了下来。

（配图：龙门浩老街牌匾 / 集市热闹场景 / 老街街巷随拍）

中午，我们返回渝中区的酒店收拾行李，转移阵地——换到了奥体中心旁边的汉庭酒店，离体育馆近得离谱，步行几步就能到达，为晚上的演唱会省去了不少麻烦。

（配图：汉庭酒店门口 / 酒店到奥体中心的步行路景）

# 黑马之约：旋律里的青春与热爱

夜幕降临，我们怀着满心期待，走进了重庆奥体中心，赴这场期待已久的李荣浩“黑马”巡回演唱会之约。演唱会开篇便惊艳全场，舞台灯光骤然暗下，前奏响起，正是专辑同名主打歌《黑马》，李荣浩缓缓走上舞台，眼睛上蒙着一条红色丝带，简约又帅气，自带气场，全场瞬间沸腾，荧光棒的光芒汇聚成一片星海，欢呼声、呐喊声此起彼伏。

（配图：李荣浩蒙红色丝带登场特写 / 全场荧光海远景 / 舞台开篇灯光）

“黑马不是不合群，是听天不由命”，当这句歌词响起，我瞬间被击中，这不仅是歌曲的主旨，更是我们青春里最动人的韧劲，不迎合、不妥协，带着热爱向阳生长，想起自己一路以来的坚持，与这句歌词产生了深深的共鸣，忍不住跟着大声合唱。

（配图：歌词大屏特写 / 我们跟着合唱的随拍）

紧接着，《模特》的前奏缓缓响起，节奏舒缓又带着几分慵懒，“穿华丽的服装，为原始的渴望而站着”，李荣浩的嗓音低沉有磁性，将歌曲里的孤独与清醒演绎得淋漓尽致，全场瞬间安静下来，只剩下他的歌声和轻轻挥舞的荧光棒，我闭上眼睛，沉浸在旋律里，仿佛看到了那些在生活中努力伪装、却藏着本心的人们。

（配图：李荣浩演唱《模特》特写 / 舞台温柔灯光 / 荧光棒轻挥场景）

随后，《乌梅子酱》的旋律响起，瞬间将氛围拉向温柔，“你浅浅的微笑，就像乌梅子酱”，甜甜的歌词搭配他温柔的唱腔，连空气都变得甜腻起来，身边的观众都跟着轻轻哼唱，我转头看向身边的好友，两人相视一笑，所有的美好都藏在这温柔的旋律里，想起那些细碎的欢喜时光，满心都是暖意。

（配图：《乌梅子酱》歌词大屏 / 我们相视一笑的合照 / 舞台甜美元气灯光）

当《老街》的旋律响起，我的眼眶瞬间湿润了，白天在龙门浩老街的所见所感瞬间涌上心头，“巷尾老爷爷卖的热汤面，味道弥漫过旧旧的后院”，熟悉的歌词，温柔的旋律，唱尽了对旧时光的怀念，也勾起了我高二时游山城的细碎回忆，那些藏在青春里的温暖与遗憾，都在这歌声里慢慢释怀。

（配图：李荣浩演唱《老街》特写 / 龙门浩老街旧景呼应照 / 荧光棒温柔挥舞）

除了这些温柔的抒情曲，李荣浩也带来了很多燃炸全场的歌曲，鼓点一响，全场瞬间被点燃。他带领着全场观众站起来欢呼、跳跃，挥舞着荧光棒，大声合唱，那种热血沸腾的感觉，难以用语言形容。中途的打鼓环节更是惊艳全场，他褪去吉他，走到架子鼓前，拿起鼓棒，全身心投入其中，鼓点铿锵有力，节奏越来越快，每一下都敲在人心上，到最后，他索性丢掉鼓棒，用手直接敲击鼓面，化身“爆裂鼓手”，那份肆意与热爱，感染着现场的每一个人。

（配图：李荣浩打鼓特写 / 全场站立欢呼的场景 / 丢掉的鼓棒特写）

之后，他演唱了自己为别人创作的经典作品——莫文蔚的《慢慢喜欢你》，“慢慢喜欢你，慢慢的亲密，慢慢聊自己”，温柔的歌词经他重新演绎，多了几分独特的“浩式”韵味，没有原唱的柔美，却多了几分沉稳与深情，全场再次陷入大合唱的热潮，我跟着旋律轻轻哼唱，感受着这份慢慢沉淀的温柔与爱意。

（配图：李荣浩演唱《慢慢喜欢你》特写 / 全场大合唱背影 / 温柔灯光氛围）

紧接着，《丑八怪》的旋律响起，节奏瞬间变得明快，“丑八怪，能否别把灯打开，我要的爱，出没在漆黑一片的舞台”，他用有力的唱腔，唱出了歌曲里的挣扎与坦荡，台下的观众热情高涨，跟着节奏挥舞荧光棒，呐喊声不绝于耳，我也跟着大声唱，把所有的情绪都释放出来，格外畅快。

（配图：《丑八怪》歌词大屏 / 全场热情欢呼特写 / 舞台炫酷灯光）

更惊喜的是，他坐在吊机上，像荡秋千一样缓缓升起，身后是璀璨的灯光和巨型屏幕，仲夏夜般的舞台，温柔又浪漫，他抱着吉他，轻声演唱着《紫荆花盛开》，“紫荆花盛开，像你的裙摆，轻轻摇摆，把温柔都散开”，温柔的旋律搭配治愈的歌词，那一刻，仿佛所有的美好都定格在了眼前，场馆里一片安静，所有人都沉浸在这温柔的氛围里。

（配图：吊机上的李荣浩 / 仲夏夜风格舞台 / 《紫荆花盛开》歌词特写）

整场演唱会，时长两个多小时，没有多余的废话，全是满满的诚意，每一首歌曲，每一个舞台设计，每一次互动，都能感受到他的用心。他真的是一位全能音乐人，编曲、奏乐、演唱、作词，每一项都做到了极致，舞台中央的巨型黑马装置，搭配曲面大屏和炫酷灯光，随着旋律的起伏不断变换光影，氛围感拉满。

（配图：巨型黑马装置特写 / 舞台光影变换 / 李荣浩舞台互动瞬间）

演唱会结束的那一刻，灯光亮起，他鞠躬致谢，全场观众大声喊着“李荣浩”，舍不得他离开，那种意犹未尽的感觉，萦绕在心头。走出场馆，耳边依旧回荡着他的歌声，手里的荧光棒还在闪烁，那种热血与感动，久久没有散去。这场演唱会，不仅是一场视听盛宴，更是一场与热爱的双向奔赴，是青春里最珍贵、最难忘的回忆。

（配图：李荣浩鞠躬致谢的特写 / 走出场馆的人群 / 我们手里的荧光棒）

# 尾声：

演唱会结束的第二天下午，我们踏上了返回深圳的飞机。飞行途中，幸运地邂逅了一场绝美日落——太阳悬在远方，我们穿梭在云层之上，即将穿越云层的瞬间，整片云海都被染成了温柔的橙色，像被熔金包裹，静谧又璀璨，美得让人移不开眼。

（配图：飞机上的橙色云海日落 / 返程机票 / 旅行合照）

这场山城之行，短暂却无比充实。它不是一场刻意的打卡之旅，而是一场以歌为契、以情怀为名的重遇。

于我而言，山城是藏着旧回忆的地方，高二的细碎时光，在重走的街巷里慢慢浮现；于我们而言，山城是盛满热爱的地方，李荣浩的旋律，朋友的故乡情怀，让这座山城更具温度。

我喜欢山城的魔幻地形，喜欢它雨夜的浪漫，喜欢它的市井烟火，喜欢它藏在细节里的温柔；更喜欢这场旅行里的每一份惊喜与感动——是旧忆重逢的感慨，是好友相伴的惬意，是奔赴热爱的热血，更是山城独有的治愈力量。

山城的风，吹过了我们的青春与热爱，这场以歌为约的旅程，终将成为我们记忆里最珍贵的片段。山城，我们后会有期。



（撰稿：2025/11/05）
