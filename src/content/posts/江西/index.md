# 赣地行记：南昌烟火与景德镇瓷韵

1月13号正式放假，在家只休整了两天，15号下午便匆匆出发，前往江西。这趟旅程不算长，却一头扎进了南昌的厚重与景德镇的温柔里。

傍晚抵达南昌，城市已经慢慢入夜。街道宽敞，晚风微凉，一切都显得安静又舒展，让人对接下来的行程充满期待。
（配图：南昌城市夜景 / 抵达随拍）

## 南昌：烟火与历史交织的英雄城
第二天在南昌慢慢逛开。先是去了江西省博物馆，一件件文物静静陈列，从远古到近代，江西的文脉与历史在眼前缓缓铺开，看得人心静又感慨。
（配图：江西省博展品 / 馆内场景）

之后走到摩天轮脚下，巨大的轮体静静矗立，衬着天空格外醒目。再到八一广场，庄严开阔，处处透着这座英雄城市的气质。
（配图：南昌摩天轮 / 八一广场）

傍晚钻进万寿宫，才算真正感受到南昌的烟火气。街巷热闹，小吃飘香，人来人往，人声鼎沸，老建筑与新潮小店挤在一起，热闹又鲜活。
（配图：万寿宫街巷 / 小吃街随拍）

第三天专程去了滕王阁。登楼远眺，江水悠悠，楼宇层叠，“落霞与孤鹜齐飞”的意境虽未全见，却也能体会到古人登临时的心境。历史的厚重与江风一起扑面而来。
（配图：滕王阁外观 / 登楼远眺）

下午便离开南昌，前往景德镇。从一座城到另一座城，从厚重历史，转向瓷都的温柔诗意。

## 景德镇：与瓷器相遇的浪漫夜晚
抵达景德镇已是晚上，第一站直奔陶溪川。一排排创意小摊整齐排列，瓷器、手作、小摆件琳琅满目，灯光一打，每一件都精致又好看。慢慢逛着，像走进了一个陶瓷的梦境。
（配图：陶溪川夜景 / 创意小摊）

第四天一早去了陶阳新村，这里更是瓷器的世界。街边小店一家接一家，各种陶器、瓷艺琳琅满目，看得人眼花缭乱。我挑了三瓶精致的瓷花，又给朋友们选了不少小纪念品，收获满满。
（配图：陶阳新村小店 / 瓷花与纪念品）

在景德镇的日子节奏很慢，没有匆忙赶路，只有随处可见的瓷片、陶艺与烟火，整座城市都透着温润的质感。

1月19日，踏上返回深圳的路程。
从南昌的庄重烟火，到景德镇的瓷韵温柔，江西之行短暂却充实。
在洋楼林立、海河晚风里，感受独属于津门的松弛与浪漫。而赣地之行，则让我在历史文脉与瓷艺烟火中，收获了另一番平静与治愈。
